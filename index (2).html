SDG 12 : Responsible Consumption And Production
Introduction

This report documents the completion of five Android application development labs, covering essential concepts such as user interface design, user interaction, Material Design implementation, screen navigation, and local data persistence using Room. Each lab built progressively on the previous one, contributing to a working understanding of how a complete mobile application is structured and developed. These foundational skills support the design of an app that is functional, user-friendly, and aligned with good design practices.

Labs

01

Lab 1 (UI Design)

Introduction to Android UI design using layouts and widgets.

Reflection: This lab introduced the fundamentals of Android user interface design, covering LinearLayout, ConstraintLayout, and basic widgets. I learned how XML connects with activity code and how attributes control layout structure. This strengthened my foundation in building organized interfaces.

02

Lab 2 (User Interaction)

Handling user input and event-driven programming.

Reflection: This lab focused on event listeners such as onClickListener and handling EditText input. I learned how to pass data between activities using Intents. It improved my understanding of interactive app behaviour.

03

Lab 3 (Material Design)

Applying modern UI components and theming.

Reflection: This lab demonstrated Material Design components like cards and themed buttons. I learned how design systems improve consistency and usability. It gave me a better appreciation of UI/UX principles.

04

Lab 4 (Navigation)

Fragment navigation using Navigation Component.

Reflection: This lab introduced navigation graphs and fragment transitions. I learned how structured navigation simplifies multi-screen apps. It improved my understanding of app architecture.

05

Lab 5 (Room Database)

Local data storage using Room persistence library.

Reflection: This lab introduced entities, DAOs, and database classes. I learned how Room simplifies SQL operations with compile-time safety. It helped me understand persistent local storage implementation.
